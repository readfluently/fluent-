Dialogue

Write a conversation clearly using speaker names, colons, and a new line for each speaker.

Writing guide

Plan who is speaking, what they want to say, and how the conversation should flow.

Purpose

To record a conversation directly from the speakers’ point of view.

Text structure
  • Write the speaker’s name on the left
  • Use a colon after the name
  • Use a new line for each speaker
  • Optional action notes in brackets
Language features
  • Can be casual or more polite depending on speakers
  • Use natural-sounding speech
  • Use requests, questions, answers, and reactions clearly
Task

Write a dialogue between two learners planning a class reading event.

Helpful starter

Naledi: Are you ready to plan the reading event?
Musa: Yes, but we still need to choose the books.

Checklist
  • Did I use a new line for each speaker?
  • Did I use names and colons correctly?
  • Does the dialogue sound natural?
  • Does the conversation make sense from start to end?

Teacher-style feedback

    Exemplar dialogue

    Naledi: We need to finish planning the reading event today. Musa: I know. Have we chosen the books yet? Naledi: Not yet. I think we should choose one storybook and one information book. Musa: That sounds good. Who will read the storybook aloud? Naledi: I can do that. Can you make the poster for the event? Musa: Yes, I can. What time should the event start? Naledi: Let’s start at two o’clock after school. Musa: Great. Then we just need to ask the teacher for permission. Naledi: Perfect. I think our plan is ready.
